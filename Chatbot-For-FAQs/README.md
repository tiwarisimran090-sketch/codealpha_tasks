# 🤖 Intelligent FAQ Support Chatbot

An AI-powered automated customer support agent that utilizes Natural Language Processing (NLP) to parse user inputs and match them with appropriate responses.

## 🧠 NLP Pipeline Mechanics
- **Text Tokenization & Cleaning:** NLTK (Natural Language Toolkit)
- **Vector Model:** Scikit-Learn TF-IDF Vectorization
- **Intent Matching Engine:** Mathematical Cosine Similarity scoring layers

## 🛠️ Tech Stack
- **Backend Core:** Python, Flask, Flask-CORS
- **Frontend UI:** HTML5, CSS3 (Premium Dark Mode UI), JavaScript Fetch APIs
